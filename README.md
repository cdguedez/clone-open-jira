# APP OPEN JIRA WITH NEXTJS

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

## How to use

```bash
# Clone this repository
$ git clone
# Go into the repository
$ cd app-open-jira-with-nextjs
# Install dependencies
$ npm i
# Run the app
$ npm run dev
```

## run docker image mongo db

```bash
# Run docker image mongo db
$ docker-compose up -d
```
